Information

    A graphical model or probabilistic graphical model (PGM) is one general paradigm to describe the conditional dependence between random variables. Applications of graphical model include topic models, computer vision and genetics. In this reading group, we have an investigations on PGM from both mathematical and algorithmic perspectives. We will also discuss some applications of PGM if time permits. Topics will include:

  • Basic graph theory, graphical model representation
  • Exact inference and approximation inference
  • Learning algorithms for graphical model
  • Popular graphical model and nonparametric Bayesian models
  • Spectral methods for graphical model
  • Structured sparsity
  • Scalable algorithms for graphical model
